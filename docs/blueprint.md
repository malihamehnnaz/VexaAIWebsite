# **App Name**: Vexa AI

## Core Features:

- Service Catalog: Display a catalog of AI, software development, and cloud services with detailed descriptions and pricing.
- AI-Powered Recommendation: Suggest suitable service packages based on the user's specified needs and business profile; incorporate a tool to help the user choose an appropriate service.
- Client Portfolio: Showcase successful projects and client testimonials to build trust and credibility.
- Contact Form: Allow potential clients to send inquiries.
- About Us Section: Detail company's mission, vision, and values.
- Blog: Publish informative content on AI, software development, and cloud services.
- AI Chatbot: Provide a chatbot powered by gen AI to answer customer questions

## Style Guidelines:

- Primary color: Deep navy blue (#1E3A8A) to convey professionalism and trust.
- Background color: Light gray (#F7F7F7) for a clean and modern feel.
- Accent color: Muted teal (#558B96) to add a touch of sophistication.
- Body font: 'PT Sans', sans-serif, for readability and a modern look; headlines: 'Space Grotesk', sans-serif, for a high-tech feel.
- Use clean, minimalist icons to represent services and features.
- Use a grid-based layout for a structured and organized design.
- Incorporate subtle animations and transitions to enhance user experience.
